Projects

Select side projects and work I've done over the past.

Mapping Football Stadiums in the U.K

An interactive map with filters built using Leaflet.js and Chart.js that gives a high level visualization of the spread of football stadiums in the United Kingdom. Interesting clusters of stadiums can be noted around select urban areas.

The Lunch App

Quick search tool created to help find places to grab lunch nearby. Uses the FourSquare API and HTML5 geolocation to display interesting nearby restaurants to explore.

SteamMaku

SteamMaku is a 2-dimensional competitive bullet hell game that is to be played with two platers. Built using OCaml.
